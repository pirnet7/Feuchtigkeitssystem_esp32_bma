# Feuchtigkeitssystem ESP32 – BMA

Dieses Projekt ist Teil unserer **Berufsmaturitätsarbeit (BMA)**.
Es handelt sich um ein **IoT-basiertes Feuchtigkeitssystem für Topfpflanzen**.

Ein **ESP32** misst die Bodenfeuchtigkeit einer Pflanze, sendet die Daten über **MQTT** und zeigt sie in einer **Flutter-App** an.
So erkennt man, wann eine Pflanze zu trocken ist.

---

## Funktion

- Messung der Bodenfeuchtigkeit mit einem Sensor
- Übertragung der Messwerte über WLAN (MQTT)
- Anzeige der Werte in einer App
- Warnung bei zu trockener Erde

---

## Verwendete Technologien

- ESP32 (M5Stack)
- Bodenfeuchtesensor (Earth Unit)
- MQTT (z. B. shiftr.io)
- Flutter App

---

## Ziel des Projekts

Das Ziel ist es, eine **einfache Lösung zur Überwachung der Bodenfeuchtigkeit** von Zimmerpflanzen zu entwickeln und die Messwerte verständlich darzustellen.
